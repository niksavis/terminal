# Cheat Sheet Publishing

Maintainer-focused publishing notes for the terminal cheat sheet.

- Live page: <https://niksavis.github.io/terminal/>
- Source: terminal-cheat-sheet.md
- Build script: .scripts/render-cheat-sheet.py (renders terminal-cheat-sheet.html)
- Workflow: .github/workflows/cheat-sheet.yml (render + GitHub Pages deploy on main)
- One-time setting: GitHub repository Settings > Pages > Source = GitHub Actions
